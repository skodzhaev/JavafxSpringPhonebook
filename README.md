Phonebook
===============

Простая телефонная книга. Поддерживается поиск и проверка на совпадение имен.
Больше ничего.

Использованные инструменты
------------

Сделано с помощью Java SDK - https://vk.com/dev/Java_SDK 
Интерфейс построен при помощи javafx + Gluon Scene Builder 8.3.0
База данных - h2 1.4.193
ORM - Hibernate 5.2.8
IoC container - Spring 4.3.3

Зачем все это, что за куча технологий для простой программы?
------------
Проект был сделан для проверки совместимости стандартного IoC контейнера спринга
и FXML контейнера, без использования Spring Boot и подобных. Получилось так себе.
